## Core Equations

### Affine-Invariant Riemannian Metric

For symmetric positive-definite (SPD) covariance matrices `Sigma_1, Sigma_2 in P(n)`, the affine-invariant distance is:

```
d(Sigma_1, Sigma_2) = || log(Sigma_1^{-1/2} ┬╖ Sigma_2 ┬╖ Sigma_1^{-1/2}) ||_F
```

### Tangent Space Projection

Map a covariance matrix `Sigma_t` to the tangent space at `Sigma_ref`:

```
T_t = Sigma_ref^{1/2} ┬╖ log(Sigma_ref^{-1/2} ┬╖ Sigma_t ┬╖ Sigma_ref^{-1/2}) ┬╖ Sigma_ref^{1/2}
```

The tangent space `T_P(n)(Sigma_ref)` is the space of symmetric n├Чn matrices. The inverse map is:

```
Sigma_t = Sigma_ref^{1/2} ┬╖ exp(Sigma_ref^{-1/2} ┬╖ T_t ┬╖ Sigma_ref^{-1/2}) ┬╖ Sigma_ref^{1/2}
```

### Vectorization

Only the upper triangle (including diagonal) is transmitted as feature vector:

```
v_t = vech(T_t)  тИИ  R^{n(n+1)/2}
```

Weighting: diagonal elements at full scale, off-diagonal at sqrt(2) scale to preserve the Frobenius norm.

### Frechet Mean (Offline Reference)

Given a set of covariances `{Sigma_i}`, the Frechet mean is the minimizer of the sum of squared Riemannian distances:

```
Sigma_ref = argmin_{Sigma in P(n)} sum_i d(Sigma, Sigma_i)^2
```

Computed via iterative gradient descent on the manifold:

```
for k = 1..K:
    T = (1/N) ┬╖ sum_i log(Sigma_k^{-1/2} ┬╖ Sigma_i ┬╖ Sigma_k^{-1/2})
    Sigma_{k+1} = Sigma_k^{1/2} ┬╖ exp(T) ┬╖ Sigma_k^{1/2}
```

Converged in < 10 iterations for n=8 (empirical).

### Geodesic Update (Online Drift Tracking)

At each frame `t`, the reference mean is updated with an entropy-dampened step:

```
eta_t = 0.005 ┬╖ max(0, 1 - H_t)
```

where `H_t` is the rolling mean Shannon entropy of model confidence over the last 0.5 seconds. During high-uncertainty periods, adaptation slows to prevent drift into noisy regions.

The update step is a single SGD step on the manifold:

```
Sigma_ref тЖР Sigma_ref^{1/2} ┬╖ exp(eta_t ┬╖ Sigma_ref^{-1/2} ┬╖ T_t ┬╖ Sigma_ref^{-1/2}) ┬╖ Sigma_ref^{1/2}
```

where `T_t` is the tangent space projection of the current `Sigma_t` at the current `Sigma_ref`.

### Procrustes Cross-Session Alignment

For loading a saved profile into a new session:

1. Collect anchor covariances from both sessions: `{P_i}` (saved) and `{Q_i}` (new)
2. Flatten both to tangent vectors at `I`: `Y_saved[i] = vech(log(P_i))`, `Y_new[i] = vech(log(Q_i))`
3. Center: subtract the mean tangent vector of each set
4. Compute optimal rotation: `[U, S, V] = svd(Y_saved_centered^T ┬╖ Y_new_centered)`, `R = U ┬╖ V^T`
5. Scale: `s = trace(S) / ||Y_new_centered||_F^2`
6. Apply to incoming frame: `v_aligned = s ┬╖ v_raw ┬╖ R`
