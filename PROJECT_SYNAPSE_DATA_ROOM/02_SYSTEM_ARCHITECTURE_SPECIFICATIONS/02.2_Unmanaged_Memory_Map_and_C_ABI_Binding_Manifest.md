# Unmanaged Memory Map & C ABI Binding Manifest — Project Synapse

## Native Library: `libbci_core.so` / `bci_core.dll`

### Source Location
`native/src/bci_core.cpp` — single translation unit, C++17, no external dependencies beyond libc.

### Compiled Library Sizes

| Platform | Library | Size (Release) |
|----------|---------|----------------|
| Windows x64 | `bci_core.dll` | ~28 KB |
| Linux x64 | `libbci_core.so` | ~24 KB |
| macOS ARM64 | `libbci_core.dylib` | ~22 KB |

### Exported C ABI Functions

```c
// Allocate and initialize a streaming covariance engine.
// Returns an opaque pointer to a CovarianceState struct.
// n: number of channels (must match CSP output dimension)
// alpha: forgetting factor (0.0–1.0, typically 0.01–0.05)
// gamma: shrinkage intensity (0.0–1.0, typically 0.05)
CovarianceState* create_covariance_engine(int n, double alpha, double gamma);

// Perform a rank-1 Cholesky update with Givens rotation sweep.
// state: opaque pointer from create_covariance_engine
// x: input array of length n (flat, row-major)
// weight_modifier: scalar multiplier for alpha (0.0 = freeze, 1.0 = normal)
void rank1_update_cholesky(CovarianceState* state, const double* x, double weight_modifier);

// Reconstruct the shrunk covariance matrix from the Cholesky factor.
// out_sigma: pre-allocated output array of length n*n (column-major flat)
void get_shrunk_covariance(CovarianceState* state, double* out_sigma);

// Free all memory associated with a covariance engine.
void destroy_covariance_engine(CovarianceState* state);
```

### Internal Memory Layout: `CovarianceState`

```c
struct CovarianceState {
    int n;              // Channel count (immutable after creation)
    double alpha;       // Exponential forgetting factor
    double gamma;       // Ledoit-Wolf shrinkage intensity
    double* L;          // Lower-triangular Cholesky factor, flat column-major array[n*n]
};
```

**Allocation lifetime:** `create_covariance_engine` → (N calls to `rank1_update_cholesky` + `get_shrunk_covariance`) → single `destroy_covariance_engine`.

### Unity P/Invoke Binding (C#)

File: `unity/BCINativeBridge.cs`

```csharp
[DllImport("bci_core", CallingConvention = CallingConvention.Cdecl)]
private static extern IntPtr create_covariance_engine(int n, double alpha, double gamma);

[DllImport("bci_core", CallingConvention = CallingConvention.Cdecl)]
private static extern void rank1_update_cholesky(IntPtr state, double[] x, double weightModifier);

[DllImport("bci_core", CallingConvention = CallingConvention.Cdecl)]
private static extern void get_shrunk_covariance(IntPtr state, double[] outSigma);

[DllImport("bci_core", CallingConvention = CallingConvention.Cdecl)]
private static extern void destroy_covariance_engine(IntPtr state);
```

The `BCINativeBridge` MonoBehaviour wraps these into a single `StepPipeline(rawFrame, weight)` call. Deployed as a component on any GameObject.

### Build Commands

```bash
# Windows (MSVC)
cl /O2 /LD native/src/bci_core.cpp /Febci_core.dll

# Linux (GCC)
g++ -O3 -fPIC -shared -o libbci_core.so native/src/bci_core.cpp

# macOS (Clang)
clang++ -O3 -fPIC -shared -o libbci_core.dylib native/src/bci_core.cpp
```

Drop the resulting `.dll` / `.so` / `.dylib` into the target project's native plugin directory (Unity: `Assets/Plugins/`).
